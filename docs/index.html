Features

Everything you need, nothing you don't

Drop-in replacement for the comma cloud. Point your device at your server and everything just works.

Drop-in API replacement

Mirrors official comma endpoints so openpilot and sunnypilot connect without any client-side patches. Just change the host URL.

Video ingestion and transcoding

Receives HEVC uploads from three cameras and transcodes to HLS for browser playback, with automatic codec fallback.

Live device communication

Persistent WebSocket channel per device for real-time JSON-RPC. Push configs, request uploads, set nav destinations.

Device configuration

Read and write device parameters through the API or web UI, with changes pushed to the device in real time over WebSocket.

GPS track storage

Stores route geometry as PostGIS LineStrings for spatial queries. View your drives on a map with full track detail.

Web dashboard

Browse routes, play back multi-camera video, view GPS tracks on OpenStreetMap, inspect uploads, and manage device settings.

Architecture

Simple, self-contained stack

comma device                                       this server

  athenad   ---- WebSocket (JSON-RPC) -------->   ws/hub
  uploader  ---- PUT /upload/... ------------>   storage layer  -->  FFmpeg transcoder
                                                  PostgreSQL + PostGIS
                                                       |
                                                  Next.js web UI

Backend (Go)

  • Echo HTTP framework
  • pgx + sqlc for type-safe database access
  • gorilla/websocket for device connections
  • FFmpeg for HEVC-to-HLS transcoding

Frontend (TypeScript)

  • Next.js + React
  • hls.js for multi-camera HLS playback
  • Leaflet + OpenStreetMap for GPS maps
  • TanStack Table + Virtual for log viewing

Data

  • PostgreSQL 14+ with PostGIS
  • Local filesystem storage
  • JWT device authentication
  • sqlc code generation from SQL

Quick Start

Up and running in five minutes

Prerequisites: Go 1.22+, Node.js 18+, pnpm, PostgreSQL 14+ with PostGIS, FFmpeg.

1

Create the database

createdb comma
psql comma -c "CREATE EXTENSION IF NOT EXISTS postgis;"
psql comma < sql/migrations/001_init.up.sql
psql comma < sql/migrations/002_device_params.up.sql
2

Configure environment

cp .env.example .env
# Edit .env -- set DATABASE_URL and JWT_SECRET
3

Start the server

go run ./cmd/server               # backend on :8080
pnpm install --dir web
pnpm --dir web dev                # frontend on :3000
4

Point your device

Set these on your comma device before openpilot starts:

export API_HOST="https://your-server.example.com"
export ATHENA_HOST="wss://your-server.example.com"
